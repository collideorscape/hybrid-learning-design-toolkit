# Hybrid Learning Assessment Method Cards
## Aligned Specifications

## Overview

The Assessment Method Cards provide educators with concrete, actionable strategies for implementing effective assessment methods in hybrid environments where students participate simultaneously from physical and virtual spaces. Each card presents essential information in a streamlined format aligned with the Activity Cards design system, with references to detailed resources for comprehensive implementation guidance.

## Integration and Usage Guidelines

### Canvas Integration
- Place Assessment Method Cards in the Assessment Strategy section of the Activity Sequence Canvas
- Purple color provides clear visual distinction from activity placement areas
- Size consistency (5" √ó 7") allows proper placement and handling alongside Activity Cards

### Cross-Reference System
- Assessment Cards numbered AS01-AS14 for clear identification
- Activity connections reference specific Activity Card numbers and names
- Resource references link to detailed implementation materials and templates

### Implementation Approach
1. **Activity-Driven Selection**: Choose assessments that align with planned learning activities
2. **Outcome-Driven Planning**: Select assessments that generate appropriate evidence for learning goals
3. **Equity-Focused Design**: Prioritize assessments that work equitably across participation modes
4. **Progressive Implementation**: Start with formative methods, build to summative approaches

### Usage Tips
- Begin with 1-2 assessment methods per learning sequence
- Balance formative and summative approaches throughout courses  
- Prioritize methods marked as working well with your selected activities
- Use detailed resource references for comprehensive implementation planning
- Consider dual-purpose methods for efficient assessment integration

This complete set of Assessment Method Cards provides comprehensive coverage of hybrid learning assessment approaches while maintaining design consistency with the broader Hybrid Learning Activity Designer toolkit.
